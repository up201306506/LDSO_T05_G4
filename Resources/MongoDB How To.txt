Criar uma pasta em c:/data/db
e abrir mongod.exe

Caso não queiram criar uma pasta no dir que disse podem especificar como argumento, assim:
mongod.exe --dbpath <dir path onde irá ser guardada as coisas do servidor da bd>

mongod.exe é o servidor da db, quando ele estiver a correr já se pode fazer CRUD à db

Para aceder ao servidor é necessário outro cmd a correr o mongo.exe
No mongo.exe, para ver as DB's guardadas usa-se:
show dbs

Para especificar qual das DB's queres usar:
use <nome da DB>

Para ver todas as tabelas de uma DB ( em mongo chamam-se collection ):
show collections

Para ver o conteudo de uma collection de uma DB:
db.<nome da collection>.find()

Para procurar um tuplo especifico de uma collection ( em mongo chamam-se documents ):
db.collection('<nome da collection>').findOne({"atributo na DB": <atributo que estamos a procurar>}, <funcao do que se fará com o documento>);
